# Insta Cat Bot

you can use this bot to send a random cat image from [TheCatApi](https://thecatapi.com/) to your friend.

## Required Packages

- requests

- webdriver-manager

- selenium

## Install

Install requirements:

```zsh
pip install -r requirement.txt
```

if you get _Permission denied_ error run:

```zsh
sudo pip install -r requirement.txt
```

## Edit file

If everything is fine, you can open the `instaCat.py` file and edit:

- Your username and password
- Target username

## Run

Finally you can save and run the python file:

```zsh
python3 instaCat.py
```
